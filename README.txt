This is a calculator. No clicking. No ads. No spam.
Python3: MIT License. Use Ctrl+C to halt. Thanks!

a + b   adds a and b
a - b   subtracts b from a
a * b   multiplies a and b
a ** b  gets a to the power of b
a / b   divides a over b
a // b  gets the floor of a / b
a % b   gets the remainder of a / b

(a + b) * c     parentheses are evaluated first
round(a,x)      rounds a to place value x
abs(a)          gets absolute value of a
floor(a)        gets floor of a
ceil(a)         gets ceiling of a
sqrt(a)         gets square root of a
log(a)          gets natural log of a

Trigonometric functions are also supported
Constants: pi (3.1415), e (2.7182), tau (6.2831)
'&', '|', '~', '^' are bitwise operators: and, or, not, xor
